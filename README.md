# Installation

## Getting Started
*To get started, just clone the repository *

    $ git clone https://github.com/vedanti-u/site-frontend.git


### Install yarn

    $ yarn install

---
###
### To RUN

      $ yarn dev
---
  *Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.*
